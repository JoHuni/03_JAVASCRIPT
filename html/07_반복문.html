for문

        - 프로그래밍의 흐름을 제어하는 제어문 중 하나로
        특정 코드를 반복할 때 사용하는 구문
    
        - 끝이 정해져 있는 반복에서 주로 사용
        (ex. n번 반복) 

        [작성법]

        for(초기식 ; 조건식 ; 증감식){
            조건식이 true인 경우 반복 수행할 코드;
        }

        초기식 : for문을 제어하는 용도의 변수 선언

        조건식 : for문의 반복 여부를 지정하는 식
        - true 일 때 반복, false일 때 반복 종료
        - 보통 초기식에 사용된 변수를 이용해서 조건식을 작성법

        증감식 : 초기식에 사용된 값을 변환 시키는 식
          -> 변환된 초기식에 의해서 조건식의 결과가 달라지게 한다

      while 문

              - 끝이 정확하게 정해져 있지 않은 반복에서 주로 사용
        (ex : 어떤 값이 입력되기 전 까지 반복)
    
        [작성법]
    
        while(조건식){
          조건식이 true인 경우 반복 수행할 코드;
        }
    
        - 조건식이 true인 경우 내부 코드를 1회 반복 후
          다시 조건식 검사 진행

      메뉴 주문하기

      1. 김밥 : 3000
      2. 라면 : 3500
      3. 돈까스 : 5000

      while을 for문 처럼 사용하기
